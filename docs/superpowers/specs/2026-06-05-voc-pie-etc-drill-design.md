# VOC 파이차트 기타 드릴다운 패널 — 설계 문서

**날짜:** 2026-06-05  
**상태:** 승인됨

---

## 배경

VOC 대분류 파이차트에서 "기타"는 11위 이하 카테고리를 합산한 값이다. 현재는 기타가 얼마나 많은지만 보이고 어떤 항목이 포함됐는지 알 수 없어 불편하다.

## 목표

파이차트의 `기타` 슬라이스를 클릭하면 우측에 세부 항목 테이블 패널이 나타난다. 다시 클릭하면 닫힌다.

---

## 동작 명세

1. 사용자가 파이차트에서 `기타` 슬라이스를 클릭한다.
2. 차트 오른쪽에 "기타 세부 항목" 패널이 슬라이드인으로 표시된다.
3. 패널 안에 순위·대분류·건수 테이블이 표시된다 (건수 내림차순).
4. `기타` 슬라이스를 다시 클릭하면 패널이 닫힌다 (토글).
5. 다른 슬라이스를 클릭해도 패널은 닫히지 않는다.

---

## 데이터 흐름

```
renderVoc()
  └─ cat1Sorted.slice(VOC_PIE_TOPN)  →  etcDetails (배열: [[대분류, 건수], ...])
       └─ drawPie('voc-pie', cat1A, 'vocDistChart', { pctInside: true, etcDetails })
            └─ Chart.js onClick 이벤트
                 └─ 클릭된 라벨 === '기타' → #voc-etc-panel 토글
```

## 변경 파일

### `docs/app.js`

1. **`renderVoc()`**: `cat1Sorted.slice(VOC_PIE_TOPN)`를 `etcDetails` 변수로 저장, `drawPie()` opts에 전달.
2. **`drawPie()`**: `opts.etcDetails`가 있을 때 Chart.js `onClick` 핸들러 등록. 클릭된 슬라이스 라벨이 `'기타'`면 `#voc-etc-panel` 토글.
3. **패널 렌더 함수 `renderEtcPanel(etcDetails, containerId)`**: `etcDetails`를 받아 테이블 HTML 생성 후 패널에 삽입.

### `docs/index.html`

- 파이차트 패널(`pie-wrap`) 안에 flex 컨테이너 추가.
- `<canvas id="voc-pie">` 와 나란히 `<div id="voc-etc-panel" hidden>` 추가.

### `docs/style.css`

- `#voc-etc-panel`: 최소 너비 220px, 테이블 스타일 (하단 voc-table과 동일 계열).
- `.pie-wrap`: `display: flex`, `gap: 16px`, `align-items: flex-start`.

## 비고

- 기타 슬라이스가 없을 때(11위 이하 카테고리 없음)는 클릭해도 패널이 열리지 않는다.
- 채널(전체/채팅/전화) 전환 시 패널은 닫힌다 (stale 데이터 방지).
- 비교 기간 모드에서도 동일하게 동작한다.
